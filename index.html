Introducción

Reto Html & CSS

Para aprender, prácticar y poner a prueba las habilidades en HTML y CSS. Que mejor manera que realizar un sitio web, que refleje los conocimientos y habilidades que actualmente se encuentra estudiando.

Será un reto desde el principio, escoger el tema, buscar los recursos, definir el contenido, optimizar cada paso, hacer que se vea bien y que todo funcione perfectamente.

Esperamos que todos puedan alcanzar su objetivo de creación del primero de muchos sitios web, desde hoy emprendemos el desafío de lograr este primer reto, un producto profesional para agregar a su portafolio como entregable del diplomado de fullstack.
